## Client-Side Navigation (Fast)

To do client-side navigation, you need to use the `Link` component provided by Next.js.

First, modify `pages/index.js` as follows to use the `Link` component:

```jsx
import Link from 'next/link';

const Index = () => (
  <div>
    <h1>Index</h1>
    <p>
      <Link href="/about">
        <a>About</a>
      </Link>
    </p>
  </div>
);

export default Index;
```

**Note:** The `href` attribute was moved to `Link` from the `a` tag.

Then, modify `pages/about.js` as follows:

```jsx
import Link from 'next/link';

const About = () => (
  <div>
    <h1>About</h1>
    <p>
      <Link href="/">
        <a>Index</a>
      </Link>
    </p>
  </div>
);

export default About;
```
