## Blog Post Information

First, let’s create a file to store information about each blog post:

- Create the `lib` directory (on the top level of the project folder)
- Add `posts.js` inside with the following contents:

```js
export const postsData = {
  'first-post': {
    title: 'First Post',
    date: new Date('2020-01-01')
  },
  'second-post': {
    title: 'Second Post',
    date: new Date('2020-01-02')
  }
};

// postsData keys sorted from the newest to the oldest
// ['second-post', 'first-post']
export const postsKeysSorted = Object.keys(postsData).sort(
  (a, b) => postsData[b].date - postsData[a].date
);
```

- `postsData` contains data about each blog post. It stores post data such as `title` and `date`.
- `postsKeysSorted` is a sorted list of `postsData` keys. It’ll be ordered from the newest to the oldest based on `date`.

### Display the post names on the index page

Let’s display the post names on the index page. Modify `pages/index.js` as follows:

```jsx
import Link from 'next/link';
import { postsData, postsKeysSorted } from '../lib/posts';

const Index = () => (
  <div>
    <h1>Index</h1>
    <ul>
      {postsKeysSorted.map(key => (
        <li key={key}>{postsData[key].title}</li>
      ))}
    </ul>
    <p>
      <Link href="/about">
        <a>About</a>
      </Link>
    </p>
  </div>
);

export default Index;
```

If you open [http://localhost:3000](http://localhost:3000), you should now see:

- Second Post
- First Post
