## Hot Reloading

Let’s try editing the page. In `pages/index.js`, change what’s inside the `<h1>` tag to some random text:

```jsx
const Index = () => <h1>Test Test</h1>;
```

If you save the file, it _automatically_ updates the browser with the new text. This is the <strong>hot reloading</strong> feature of Next.js.

### What happens if you forget `export default`?

Let’s experiment for a bit and see what happens if you forget to `export default` the component. Comment out the last line like so, and save the file:

```jsx
const Index = () => <h1>Hello Next.js</h1>;

// export default Index
```
