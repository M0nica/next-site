## Add MDX

First, rename `pages/posts/first-post.js` to `first-post.mdx`.

Then, modify `first-post.mdx` as follows:

```md
import PostLayout from '../../components/PostLayout';

export const title = "First Post"

This is the first blog post.

**Technologies used:**

- Next.js
- CSS Modules
- MDX

export default ({ children }) => <PostLayout title={title}>
{children}
</PostLayout>;
```

`import` and `export` are similar to before, but everything in between is now in Markdown. As you can see, MDX lets you mix JSX and markdown. You can learn more about MDX on [their documentation](https://mdxjs.com/).

If you save the file, the page should now look like this:
